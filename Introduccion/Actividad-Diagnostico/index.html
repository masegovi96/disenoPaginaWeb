Actividad Diagnóstica

HTML y CSS — Landing Page

Evalúa tu dominio en HTML y CSS creando una landing page sobre un tema de tu interés.

Importante

  • Esta actividad no tiene peso en la calificación final — su propósito es únicamente diagnóstico.
  • Evita utilizar herramientas de Inteligencia Artificial — el objetivo es conocer tu nivel real de conocimientos.

Instrucciones Generales

Desarrolla una landing page de una sola página sobre un tema libre de tu elección.

Puedes elegir cualquier tema, por ejemplo:

Videojuegos
Música
Tecnología
Deportes
Series
Cualquier otro
Siempre y cuando sea un tema apropiado y respetuoso

Estructura Mínima

1. Header

  • Nombre del sitio o logo
  • Menú de navegación (anclas internas)

2. Hero

  • Título llamativo
  • Subtítulo o descripción breve
  • Botón
  • Imagen representativa

3. Contenido

Sobre el tema elegido

  • Texto descriptivo
  • Imágenes
  • Lista o información organizada

4. Tarjetas

Mínimo 3 elementos, cada uno con:

  • Título
  • Descripción
  • Imagen o ícono

5. Contacto

Formulario (estructura HTML) con:

  • Nombre
  • Correo
  • Mensaje

6. Footer

  • Información básica
  • Redes sociales

Requisitos de CSS

Maquetación

  • Uso obligatorio de Flexbox y/o Grid

Estilos

  • Colores personalizados
  • Tipografía distinta a la default
  • margin, padding, border
  • Efectos hover

Diseño Visual

  • Buena alineación
  • Espaciado consistente
  • Jerarquía clara (títulos, textos, botones)

Responsive

  • Adaptación a celular
  • Uso de media queries

JavaScript

Solo agrega validación básica del formulario (verificar que los campos no estén vacíos antes de enviar).

Repositorio (Obligatorio)

La actividad debe subirse a GitHub. Requisitos:

Crear o usar un repositorio personal
Crear una carpeta llamada Examen Diagnóstico en la raíz del repositorio y colocar ahí todos los archivos del examen
Incluir archivos HTML, CSS y JS
Invitar al profesor como colaborador del repositorio para poder revisar el código
