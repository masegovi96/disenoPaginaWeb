Ciclo Escolar

Bienvenidos alumnos de 3er Cuatrimestre

Esta página servirá como nuestro repositorio y guía principal para la materia de Diseño de Página Web. Aquí encontrarás apuntes, tareas, proyectos y recursos importantes para tu desarrollo.

Además de subirse al Classroom, podrán encontrar todos los recursos de la clase en el GitHub.

Enfoque de la Materia

Diseño UX / UI

Aprenderemos a pensar en el usuario final. Crearemos experiencias intuitivas, accesibles y estéticamente agradables utilizando principios de Experiencia de Usuario.

HTML & CSS Moderno

Dominaremos la estructura y el estilo visual de la web, pasando por layouts con Flexbox y Grid, hasta animaciones, variables y diseño responsivo.

Frameworks (Bootstrap)

Utilizaremos herramientas estándar de la industria como Bootstrap para acelerar el desarrollo y mantener un diseño consistente y adaptable en cualquier dispositivo.

JavaScript Básico

Le daremos vida y dinamismo a nuestras interfaces web aprendiendo a manipular el DOM e interactuar con el usuario mediante JavaScript.

Criterios de Evaluación

Criterio Ponderación Aspectos a Calificar
Tareas 25% Entrega de actividades en tiempo y forma a través de Classroom y GitHub. No se aceptarán entregas extemporáneas. Incluye actividades realizadas en clase y tareas para casa. Posterior a cada entrega, el alumno deberá explicar su trabajo y responder preguntas relacionadas. Nota: Se revisarán los logs de Git para validar la hora exacta de la entrega.
Participación 25% Se asignará el puntaje total al alumno (o alumnos) con mayor número de participaciones, el resto recibirá un porcentaje proporcional a su participación. Generalmente consisten en actividades que incluyen programar, y que se realizan durante la clase.
Asistencia 5% Asistencia puntual a la sesión. El pase de lista se realiza a las 7:10 AM. Quienes lleguen entre las 7:11 AM y 7:15 AM será considerado como retardo. Después de las 7:15 AM se contabilizará como falta y no se permitirá el acceso al aula.
Uso de uniforme 5% Asistir a clase con el uniforme institucional completo y en buen estado. El incumplimiento de esta norma implicará la pérdida de este porcentaje.
Examen 40% Evaluación mixta. Incluye teoría, pero aproximadamente el 70% del valor es práctico, por lo que es indispensable demostrar dominio técnico sobre los temas vistos.

Notas importantes

Uso de inteligencia artificial: La IA es una herramienta de apoyo para el aprendizaje, pero no debe sustituir tu capacidad de razonamiento. Queda completamente prohibido el uso de IA para realizar tareas o actividades de participación de forma íntegra. Si se detecta el uso excesivo de estas herramientas en tareas o participaciones, la entrega será anulada y se calificará con 0.

Integridad en exámenes: Durante la aplicación de exámenes, el uso de cualquier IA está estrictamente prohibido. En caso de detectar a un alumno utilizándola, se procederá a retirar el examen de inmediato y la calificación será de 0.

Configuración de herramientas: Es obligatorio contar con una cuenta de GitHub y un repositorio configurado correctamente para la entrega de tareas y proyectos. El flujo de trabajo para tareas, participaciones y exámenes se gestionará exclusivamente a través de GitHub y Classroom.
